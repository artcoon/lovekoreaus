## 1. 콜드스타트 진단

양면 시장의 핵심 문제: **셀러가 없으면 바이어가 안 오고, 바이어가 없으면 셀러가 안 온다.**

| 상태 | 문제 | 해결 방향 |
|------|------|-----------|
| 셀러 0명 | 비어있는 디렉토리 → 신뢰 없음 | 운영팀이 직접 콘텐츠 시딩 |
| 바이어 0명 | 셀러에게 보여줄 트래픽 없음 | 콘텐츠(영상/SEO) 기반 유기 유입 먼저 확보 |
| 데이터 0건 | "등록 업체 2,500개" 같은 신뢰 지표 불가 | 실제 수치가 쌓일 때까지 대안 지표 사용 |

**해법 순서: 콘텐츠 먼저 → 트래픽 확보 → 셀러 유치 → 양면 효과 작동**

## 2. 콘텐츠 시딩 전략

셀러 가입 전에도 사이트가 "비어 보이지 않도록" 운영팀이 직접 채우는 초기 콘텐츠.

### 2.1 시딩 콘텐츠 유형

#### A. 유튜브 영상 큐레이션 (최우선, 비용 0)

셀러 없이도 즉시 가능. Watch 페이지와 카테고리 페이지를 영상으로 채움.

| 카테고리 | 큐레이션 키워드 예시 | 목표 수량 (런칭 전) |
|----------|---------------------|-------------------|
| K-뷰티 | "Korean skincare routine", "K-beauty review" | 50편 |
| K-푸드 | "Korean food review", "Korean snack unboxing" | 40편 |
| K-패션 | "Korean fashion haul", "K-style lookbook" | 30편 |
| K-컬처 | "K-pop merch", "Korean stationery haul" | 20편 |
| B2B/제조 | "Korean factory tour", "K-beauty manufacturing" | 20편 |
| **합계** | | **160편** |

실행 방법:
- YouTube Data API로 키워드·조회수 기반 자동 수집
- 에디터가 품질 검수 후 카테고리·태그 부여
- 각 영상에 "관련 상품 찾기" CTA 연결 (초기엔 카테고리 리스트로)

#### B. 업체/브랜드 프로필 사전 구축

공개 정보 기반으로 대표 한국 브랜드 프로필을 운영팀이 직접 생성.

| 단계 | 내용 | 목표 |
|------|------|------|
| Tier 1 | 이미 미국 진출한 대형 K-뷰티 브랜드 (조선미녀, 코스알엑스, 라네즈 등) | 30개 |
| Tier 2 | 아마존·큐텐 입점 중소 브랜드 (공개 정보 기반) | 50개 |
| Tier 3 | 수출바우처 참여 기업 목록 (공개 데이터) | 50개 |
| **합계** | | **130개** |

프로필 구성:
- 브랜드명(영/한), 카테고리, 대표 상품 3~5개(이미지는 공식 사이트 출처)
- 해당 브랜드 유튜브 리뷰 영상 2~3편 임베드
- **"이 브랜드의 공식 담당자이신가요? → 프로필 클레임하기"** CTA
- 클레임 시 셀러 가입으로 자연 전환 (LinkedIn Company Page 모델)

#### C. 매거진/에디토리얼 콘텐츠

SEO 유입 + 사이트 전문성 확보용 읽을거리.

| 시리즈 | 예시 제목 | 빈도 |
|--------|-----------|------|
| K-뷰티 가이드 | "2026 Best Korean Sunscreens for US Market" | 주 2회 |
| 바이어 가이드 | "How to Import Korean Cosmetics to the US: FDA Guide" | 주 1회 |
| 트렌드 리포트 | "K-Food Export Trends: What's Hot in 2026" | 격주 |
| 셀러 성공사례 | "How [Brand] Entered the US Market" (공개 사례 기반) | 월 2회 |
| 영상 위클리 | "This Week's Top 5 K-Beauty YouTube Reviews" | 주 1회 |

목표: 런칭 전 **20편**, 런칭 후 주 3~4편 발행

#### D. 카테고리 랜딩 페이지

8개 대카테고리 각각에 다음을 사전 배치:
- 카테고리 설명 (영문 200자 + 국문)
- 시장 규모/트렌드 요약 (기획서 조사 데이터 활용)
- 큐레이션 영상 6~8편
- 시딩된 브랜드 프로필 카드
- "이 카테고리에 입점하기" 셀러 CTA

## 3. 셀러 확보 마케팅

### 3.1 셀러 획득 채널

#### 채널 1: 프로필 클레임 전환 (핵심 전략)

시딩된 130개 브랜드 프로필에 "공식 담당자이신가요?" CTA → 클레임 시 셀러 가입.

- 예상 전환율: 10~15% (13~20개 브랜드 확보)
- 클레임 시 혜택: Pro 기능 1개월 무료, 프로필 편집 권한, 영상 추가
- 클레임 안 된 프로필은 "미인증" 라벨로 유지 → 브랜드 측의 클레임 동기 부여

#### 채널 2: 정부 수출지원사업 연계

| 지원사업 | 접근 방법 | 기대 효과 |
|----------|-----------|-----------|
| 수출바우처 (4,000개사, 1,502억원) | 수행기관 등록 or 참여기업 대상 세미나 | 바우처 예산으로 유료 입점 가능 |
| 중기부 해외마케팅 | 지방 중기센터 제휴 → 입점 설명회 | 소상공인 일괄 온보딩 |
| KOTRA 수출상담회 | 행사 부스 or 온라인 제휴 | B2B 셀러 유치 |
| 지자체 수출지원 | 경기, 충남 등 수출지원 프로그램 연계 | 지역 특산품·중소기업 |

타임라인: 수행기관 등록은 런칭 3개월 전 시작 (심사 기간 소요)

#### 채널 3: 한인 비즈니스 커뮤니티 직접 영업

| 타깃 | 접근 | 규모 |
|------|------|------|
| LA·NY 한인 상공회의소 | 이사회 프레젠테이션 + 회원사 대상 세미나 | 각 300~500개사 |
| 한인 무역협회 | 공동 행사·뉴스레터 제휴 | 1,000+ 기업 |
| K-뷰티 셀러 커뮤니티 (카페·단톡) | 무료 입점 이벤트 포스팅 | 5,000+ 셀러 |
| 아마존 한국 셀러 그룹 | 크로스 리스팅 혜택 제안 | 10,000+ 셀러 |

#### 채널 4: 콘텐츠 마케팅 (인바운드)

매거진 콘텐츠로 셀러가 먼저 찾아오게 만듦:
- "미국 FDA 화장품 등록 가이드" → CTA: "등록 완료 후 LoveKorea.Us에서 바이어 찾기"
- "2026 미국 K-푸드 시장 트렌드 리포트" → CTA: "지금 무료 입점하고 바이어에게 노출"
- "수출바우처 200% 활용법" → CTA: "바우처로 프리미엄 입점하기"

#### 채널 5: 온보딩 인센티브

| 인센티브 | 대상 | 기간 |
