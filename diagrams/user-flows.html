1. 셀러 온보딩 플로우 (Seller Onboarding)

한국 기업/소상공인이 가입 후 프로필을 완성하고 상품을 등록해 해외 노출되기까지의 전체 여정.
핵심 목표: 가입 → 프로필 완성 → 첫 상품 등록 → 노출 시작까지 30분 이내 완료. 

flowchart TD
    classDef start fill:#0B2E59,stroke:#0B2E59,color:#fff,font-weight:bold
    classDef action fill:#E8F0FE,stroke:#0B2E59,color:#0B2E59
    classDef decision fill:#FFF3E0,stroke:#E65100,color:#E65100
    classDef success fill:#E8F5E9,stroke:#2E7D32,color:#2E7D32,font-weight:bold
    classDef system fill:#F3E5F5,stroke:#7B1FA2,color:#7B1FA2
    classDef upgrade fill:#FCE4EC,stroke:#C8202F,color:#C8202F

    A["랜딩: For Sellers 페이지"]:::start
    A --> B["무료 등록하기 CTA 클릭"]:::action
    B --> C{"회원가입 방식 선택"}:::decision
    C -->|"이메일"| D["이메일 가입 + 인증"]:::action
    C -->|"소셜"| E["Google / Kakao SSO"]:::action
    D --> F["기본 정보 입력"]:::action
    E --> F

    F --> G["업체 유형 선택\n(브랜드 / 제조 / 유통 / 소상공인)"]:::action
    G --> H["사업자등록증 업로드\n(선택, 인증배지용)"]:::action
    H --> I["타깃 시장 선택\n(US / JP / CN / Global)"]:::action
    I --> J["카테고리 선택\n(Beauty / Food / Fashion 등)"]:::action

    J --> K{{"셀러 대시보드 진입"}}:::system
    K --> L["프로필 작성 가이드\n(진행률 바 0%)"]:::action

    L --> M["회사 소개 작성\n(영문 + 국문)"]:::action
    M --> N["로고 · 대표 이미지 업로드"]:::action
    N --> O["유튜브 영상 URL 등록\n(회사소개/제품리뷰)"]:::action
    O --> P["첫 상품 등록\n(이미지, 스펙, 가격/MOQ)"]:::action

    P --> Q{"프로필 완성도 ≥ 80%?"}:::decision
    Q -->|"아니오"| R["미완성 항목 알림\n+ 가이드 팝업"]:::system
    R --> M
    Q -->|"예"| S{{"운영팀 검수\n(자동 + 수동)"}}:::system

    S --> T{"검수 통과?"}:::decision
    T -->|"보완 필요"| U["보완 요청 이메일\n+ 대시보드 알림"]:::system
    U --> M
    T -->|"통과"| V["프로필 공개\n+ 검색 노출 시작"]:::success

    V --> W{"유료 전환 유도"}:::decision
    W -->|"Free 유지"| X["기본 노출 유지\n(상품 5개, 영상 1개)"]:::action
    W -->|"Pro 업그레이드"| Y["Pro 결제\n(상품 무제한, 리드 수집)"]:::upgrade
    W -->|"Premium"| Z["Premium 결제\n(브랜드관, 상단 노출)"]:::upgrade

    X --> AA["성과 리포트 수신\n(조회수, 문의수)"]:::system
    Y --> AA
    Z --> AA
    AA --> AB["지속 운영\n(상품 추가, 영상 갱신, 문의 응대)"]:::success
시작점 / 사용자 액션

2. 바이어 탐색 플로우 (Buyer Exploration)

해외 바이어/소비자가 한국 상품·업체를 발견하고 신뢰를 확인한 뒤 문의·구매로 전환하는 여정.
핵심 프레임: DISCOVER → TRUST → ACT (컨셉 삼각형 기반) 

flowchart TD
    classDef entry fill:#0B2E59,stroke:#0B2E59,color:#fff,font-weight:bold
    classDef discover fill:#E3F2FD,stroke:#1565C0,color:#1565C0
    classDef trust fill:#FFF8E1,stroke:#F9A825,color:#B8860B
    classDef act fill:#FCE4EC,stroke:#C8202F,color:#C8202F,font-weight:bold
    classDef decision fill:#FFF3E0,stroke:#E65100,color:#E65100
    classDef loop fill:#F3E5F5,stroke:#7B1FA2,color:#7B1FA2

    %% Entry Points
    E1["Google 검색\n'Korean skincare wholesale'"]:::entry
    E2["SNS / YouTube\n영상에서 유입"]:::entry
    E3["직접 방문\nlovekorea.us"]:::entry
    E4["뉴스레터 /\n지인 추천"]:::entry

    E1 --> HOME
    E2 --> HOME
    E3 --> HOME
    E4 --> HOME

    HOME["홈페이지 도착"]:::discover

    %% DISCOVER Phase
    HOME --> D1{"탐색 방식 선택"}:::decision
    D1 -->|"검색"| D2["통합 검색\n(키워드 입력)"]:::discover
    D1 -->|"카테고리"| D3["카테고리 탐색\n(Beauty, Food 등)"]:::discover
    D1 -->|"영상"| D4["Watch 섹션\n(영상 브라우징)"]:::discover
    D1 -->|"추천"| D5["홈 추천 상품\n캐러셀 클릭"]:::discover

    D2 --> D6["검색 결과 + 필터\n(시장/유형/인증/가격)"]:::discover
    D3 --> D6
    D4 --> T1
    D5 --> T3

    D6 --> D7{"결과 유형"}:::decision
    D7 -->|"업체"| T1["업체 상세 페이지"]:::trust
    D7 -->|"상품"| T3["상품 상세 페이지"]:::trust

    %% TRUST Phase
    T1 --> T2["영상 시청\n(회사소개/공장실사)"]:::trust
    T1 --> T2b["인증 배지 확인\n(FDA/HACCP 등)"]:::trust
    T1 --> T2c["리뷰 확인"]:::trust
    T2 --> T4{"신뢰 판단"}:::decision
    T2b --> T4
    T2c --> T4

    T3 --> T3a["상품 영상 시청\n(리뷰/언박싱)"]:::trust
    T3 --> T3b["스펙 · 인증 · 가격\n확인"]:::trust
    T3a --> T5{"구매 의향"}:::decision
    T3b --> T5

    %% ACT Phase - Business
    T4 -->|"신뢰함"| A1{"바이어 유형"}:::decision
    T4 -->|"더 탐색"| D1

    A1 -->|"B2B 바이어"| A2["견적 요청\n(폼 작성)"]:::act
    A1 -->|"B2B 바이어"| A3["카탈로그\n다운로드"]:::act
    A1 -->|"일반 문의"| A4["문의하기\n(이메일/폼)"]:::act

    A2 --> A7["리드 접수\n→ 셀러에게 알림"]:::act
    A3 --> A7
    A4 --> A7

    %% ACT Phase - Product
    T5 -->|"구매 의향 있음"| A5{"구매 경로"}:::decision
    T5 -->|"더 탐색"| D1

    A5 -->|"직접 구매"| A6["제휴몰 이동\n(Amazon/Coupang Global 등)"]:::act
    A5 -->|"문의/견적"| A2
    A5 -->|"위시리스트"| A8["저장 후\n나중에 구매"]:::loop

    A6 --> A9["구매 완료"]:::act
    A7 --> A10["셀러 응답 대기\n→ 거래 협의"]:::act

    A8 --> A11["리마케팅 이메일\n할인/신상품 알림"]:::loop
    A11 --> HOME
