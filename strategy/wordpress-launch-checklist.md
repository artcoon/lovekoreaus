# WordPress Launch Checklist

## Required Before First Real Post

- [ ] Change site title from `lovekoreaus` to `Love Korea US`
- [ ] Add Korean display name if desired: `러브코리아US`
- [ ] Add tagline
- [ ] Replace About page sample copy
- [ ] Remove default footer location: `123 Example Street`
- [ ] Remove default footer hours
- [ ] Remove default footer email and phone
- [ ] Add real contact email or remove contact block
- [ ] Confirm site language
- [ ] Confirm timezone
- [ ] Confirm comments policy
- [ ] Confirm whether posts should show author name

## Recommended Theme Adjustments

- [ ] Keep layout simple and readable
- [ ] Use latest posts as homepage at launch
- [ ] Add navigation links only after categories have at least one post
- [ ] Avoid empty menu sections
- [ ] Add a simple site logo later if needed

## Manual Publishing Workflow

For each post:

- [ ] Choose category
- [ ] Add 5-8 tags
- [ ] Add excerpt
- [ ] Add meta description if SEO plugin/field exists
- [ ] Add featured image
- [ ] Add image alt text
- [ ] Preview desktop
- [ ] Preview mobile
- [ ] Check links
- [ ] Publish
- [ ] Record published URL in the content calendar
