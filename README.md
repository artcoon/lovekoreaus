# Love Korea US Blog Operations

This repository stores the planning system for the WordPress blog at:

https://lovekoreaus.wordpress.com/

Current public-site status checked on 2026-06-05:

- Site title: `lovekoreaus`
- Theme: WordPress.com Assembler block theme
- Posts: none visible on the home page
- Navigation: About only
- About page: default WordPress example copy
- Footer: default example location, hours, and contact information

The immediate priority is setup before recurring content automation:

1. Finalize blog positioning and audience.
2. Replace default WordPress copy and footer placeholders.
3. Set categories, menu structure, and publishing format.
4. Publish the About page and first 3-5 foundation posts about 자유민주대한민국, 한미동맹, 역사 인식, and civic participation.
5. Start recurring content generation only after the editorial system is stable.
